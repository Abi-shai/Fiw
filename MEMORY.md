## Contexte actuel

Sitemaps Fiw (Client) et Fiw Pro (Prestataire) complétés. ADR 0003 décidé : AffiliéPartenaire dans Fiw Pro uniquement. **Deux prototypes Expo opérationnels** : `apps/fiw/` (flux Transport Client, itéré le 24 juin 2026 — nouveau flux destination → configure → searching → course-active → cloture) et `apps/fiw-pro/` (flux Mission active Transport Prestataire, complet). Architecture : deux projets Expo indépendants. Style guide validé : couleur primaire Fiw `#0066FF` (bleu), tokens dans `apps/fiw/constants/colors.ts`. **Flow Affilié Réseau breadboardé (26 juin 2026)** : vocab Ambassadeur (UI) / Affilié (recruté) / Wallet Réseau / Membre Fondateur tranché, ADR 0005 (rôle Client uniquement, « 1 affilié = 1 app »). **Flow Affilié Partenaire breadboardé (27 juin 2026)** : vocab Point Express / Wallet Partenaire / Client sauvegardé / Code prestataire tranché, ADR 0006 (Home distinct par type de compte). Nav : [Accueil][Commander][Wallet][Profil]. JS1–JS5 complets. Prochain jalon : tests utilisateurs terrain sur le flux Transport Client (`apps/fiw/`) — priorité frais de rapprochement (Options A/B/C).
